# UI Examples (Option B)

## Buttons
- Primary: bg va.blue, text va.text, hover 90%, optional subtle glow
- Secondary: border va.border, bg transparent, hover white/5%
- Ghost: transparent, text va.text2, hover white/5%

## Pills
- Draft: violet outline
- Selected: violet fill (10–15%)
- Committed: blue fill (10–15%)

## Charts
- P10 = blue, P50 = violet, P90 = magenta
